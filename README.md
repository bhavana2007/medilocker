# MediLocker

A centralized digital healthcare record system designed to securely store and manage patient medical history.

## 🌍 Problem Statement
Patients often lose physical prescriptions and medical reports, making long-term treatment tracking difficult for doctors.

## 🚀 Solution
MediLocker allows patients and doctors to upload, store, and access medical records digitally using OCR and QR-based access systems.

## ✨ Features
- Prescription Upload
- OCR Text Extraction
- QR-Based Access
- Searchable Medical History
- Secure Cloud Storage

## 🛠️ Tech Stack
- React
- FastAPI
- PostgreSQL
- Firebase Storage
- Google Vision OCR
